CSS3 -animaatiot

Teimme neljä erilaista CSS-animaatioesimerkkiä, joissa yhdistelimme erilaisia arvojen animointeja yhdeksi kokonaisuudeksi.

MediaPlayer

  • HTML
  • CSS
  • JavaScript
  • Käyttöliittymäsuunnittelu

Teimme 3 näkymään pohjautuvan MediaPlayer -sovelluksen, jonka näkymät koostuvat seuraavasti: 1. Soittolista, 2. Soivan kappaleen nimi ja taustakuva, jossa soivan albumin kansikuva, sekä YouTube-upotus videoille. Lisäksi käyttöliittymäsuunnittelun toteutimme osaltaan suunnittelemalla napit toistoon liittyen. Tämä löytyy GitHubista nimellä fuusiomediaplayer. Tein myös erikseen ulkoasultaan yksinkertaisemman MediaPlayerin, jossa play/pause/volume- napeilla on myös funktiot, tämä on ihan vain nimellä mediaplayer.
